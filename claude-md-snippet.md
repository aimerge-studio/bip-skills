## Build in Public

When any of these happen, suggest running `/bip`:

- User says "done", "that's it", "let's stop", "wrap up"
- A major feature or architecture decision is completed
- User asks "what did we do today"

Say: "Want me to log this session? Just say /bip."

Rules:

- Never ask twice in the same session
- If declined, skip silently
- Never auto-run -- only suggest
